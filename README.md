# AgroSustentável 2026 - Projeto Agrinho paranaense

> **Tema Oficial:** "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"

Este projeto é uma aplicação web interativa desenvolvida para a categoria de **Programação** do **Programa Agrinho 2026** no Estado do Paraná. O objetivo da plataforma é conscientizar produtores rurais e a comunidade sobre a importância de alinhar a alta produtividade agrícola à preservação ambiental, utilizando a tecnologia como principal ferramenta de transformação.

---

## 🚀 Funcionalidades Principais

* **Simulador de Diagnóstico Verde:** Um algoritmo em JavaScript que analisa as práticas da propriedade rural (energia, água e resíduos), calcula uma pontuação ecológica e gera recomendações técnicas personalizadas baseadas em engenharia agronômica sustentável.
* **Modo Escuro (Dark Mode):** Interface inclusiva e moderna que altera dinamicamente o tema visual do site através da manipulação do DOM (Document Object Model).
* **Design Responsivo e Fluido:** Layout moderno construído com CSS moderno (CSS Variables, Flexbox), garantindo adaptação perfeita para celulares, tablets e computadores, além de animações suaves de feedback visual (*hover effects*).

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído do zero, utilizando tecnologias nativas da web para demonstrar o domínio de lógica de programação e estruturação limpa, sem dependência de frameworks externos:

* **HTML5:** Estruturação semântica de todo o conteúdo da página.
* **CSS3:** Estilização avançada, uso de variáveis de ambiente para o Dark Mode e efeitos visuais modernos.
* **JavaScript (ES6):** Motor lógico do simulador de sustentabilidade e controle de comportamento da interface.
